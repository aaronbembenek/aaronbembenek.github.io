[AbcDatalog](http://abcdatalog.seas.harvard.edu) is an open-source implementation
of the logic programming language Datalog written in Java. It provides a basic GUI
and ready-to-use implementations of multiple standard Datalog evaluation algorithms.
It is designed to be easily extensible with new evaluation engines and language features.
